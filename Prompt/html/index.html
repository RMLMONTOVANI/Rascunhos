Distros

O que é sistema Linux?

Assim como o Windows e o MacOS, o Linux é um sistema operacional, porém com algumas peculiaridades que o torna distinto dos sistemas desenvolvidos pela Microsoft e Apple.

Open Source

Diferentemente dos demais, o Linux é livre (não necessariamente gratuito) e tem o código fonte aberto (open source), detalhe que permite aos desenvolvedores realizar modificações, otimizar, estudar, fazer testes e, até mesmo, criar distribuições próprias.

Shell

Shell é o prompt de comandos que une o usuário ao kernel, ou sistema operacional. Ele lê e interpreta as linhas de comando e dar uma resposta ao usuário através das saídas do sistema operacional.

Interface entre user e kernel

O nome Shell não foi escolhido ao acaso: semelhante a palavra “concha” em inglês, nos sistemas operacionais o shell não apenas protege o kernel de erros de usuário como também executa funções de modo seletivo. Todos os usuários podem acessar o Shell e escrever comandos para serem executados. A partir disso, a interface de comando retornará ao usuário um feedback conforme o que foi encontrado.

Distros

Uma distribuição Linux é um sistema operacional feito do kernel Linux, ferramentas do GNU, alguns programas adicionais e um gerenciador de pacotes. Com versões para servidor e desktop.

Inumeras possibilidades

As distribuições também assumem a responsabilidade de fornecer atualizações para manter o kernel e outros utilitários atualizados. Logo, o Linux em si é apenas o kernel, enquanto que as distribuições do Linux são os sistemas operacionais. Esta é a razão pela qual chamamos essas distribuições de sistemas operacionais baseados em Linux.
